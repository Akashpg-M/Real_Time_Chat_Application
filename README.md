# Real-Time Chat Application

##  Overview
This is a **real-time chat application** built using the **MERN (MongoDB, Express, React, Node.js) stack** and **Socket.io**. It supports **user authentication**, **profile updates**, **photo uploads via Cloudinary**, and **real-time online/offline status updates**.

## Features
- **JWT Authentication**: Secure login and signup using JSON Web Tokens (JWT).
- **Real-time Messaging**: Messages are sent and received in real time using **Socket.io**.
- **Online/Offline Status**: Shows which users are online or offline.
- **Profile Management**: Users can update their profile and upload photos via **Cloudinary**.
- **REST API Communication**: Uses **Express.js** with **Axios** for API requests.
- **Navigation Bar**: Includes user profile, logout, and navigation options.

##  Tech Stack
- **Frontend**: React.js, Tailwind CSS, Zustand (State Management), Axios
- **Backend**: Node.js, Express.js, MongoDB, Socket.io
- **Authentication**: JWT (JSON Web Tokens)
- **Storage**: Cloudinary (for image uploads)
